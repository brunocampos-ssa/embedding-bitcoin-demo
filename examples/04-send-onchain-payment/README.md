# 04 — Send on-chain Bitcoin

[English](README.md) | [Português do Brasil](README.pt-BR.md)

This safe mock skeleton prepares an on-chain payout and prints the review; confirm through the UI. Real sends can move funds and require a matching-network address, funded treasury, fee review, credentials, and `-tags breez`. Expected mock rail is `bitcoin`.
