# Contributing

[English](CONTRIBUTING.md) | [Português do Brasil](CONTRIBUTING.pt-BR.md)

Use Go 1.24+, Node.js 24, and npm 11. Start in mock mode with `make setup && make dev`; run `make check` before proposing a change. Keep provider types inside adapters, monetary values integral, both languages in parity, and every real payment path truthful and feature-gated. Never include secrets, real destinations, mnemonics, databases, or funded test data.

Open an issue for large design changes. Explain behavior and security impact, add deterministic tests, and update both documentation languages. By contributing you agree to the code of conduct.
