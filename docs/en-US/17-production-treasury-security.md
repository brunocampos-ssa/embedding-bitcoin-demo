# Production treasury security

[English](../en-US/17-production-treasury-security.md) | [Português do Brasil](../pt-BR/17-production-treasury-security.md)

Use a dedicated low-balance treasury, least privilege, authenticated roles, dual approval above thresholds, encrypted databases/backups, secret management, external signing, monitoring, reconciliation workers, incident response, accounting exports, and independent review. Define how pending payments are handled before any failover or retry.
