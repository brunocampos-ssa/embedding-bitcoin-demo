# Política de segurança

[English](SECURITY.md) | [Português do Brasil](SECURITY.pt-BR.md)

FreedomBounties é uma demonstração educacional, não um software de pagamentos auditado para produção. Não o use para guardar valores relevantes.

Relate vulnerabilidades em privado pelo recurso de security advisory do repositório. Inclua versão, reprodução, impacto e mitigação. Não abra issue pública para falha explorável nem teste sistemas ou fundos alheios.

Não afirmamos que a aplicação ou integração Breez tenha sido auditada. Dependências Breez têm processo e escopo próprios. A manutenção confirmará o recebimento quando possível, avaliará, coordenará a correção e dará crédito quando solicitado e seguro.
