# Como contribuir

[English](CONTRIBUTING.md) | [Português do Brasil](CONTRIBUTING.pt-BR.md)

Use Go 1.24+, Node.js 24 e npm 11. Comece em mock com `make setup && make dev`; execute `make check` antes de propor mudanças. Mantenha tipos do provedor nos adaptadores, valores monetários inteiros, os dois idiomas em paridade e rotas reais honestas e protegidas por feature flag. Nunca inclua segredos, destinos reais, mnemonic, bancos ou dados financiados.

Abra uma issue para alterações grandes. Explique comportamento e impacto de segurança, acrescente testes determinísticos e atualize os dois idiomas. Ao contribuir, você aceita o código de conduta.
